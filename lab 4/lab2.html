Справочник по HTML

1. Структура и мультимедиа

Тег / Атрибут Описание
h1 — h6 Заголовки уровней 1-6.
p Обычный абзац текста.
a Гиперссылка для перехода между страницами.
img Вставка изображения (атрибут src задает путь).
audio / video Теги для вставки звуковых и видеофайлов.
controls Атрибут, добавляющий панель управления.

2. Списки

Тег Описание
ul Маркированный (неупорядоченный) список.
ol Нумерованный (упорядоченный) список.
li Элемент списка. Используется внутри ul или ol.
dl, dt, dd Список определений (термин и его описание).

3. Формы и интерактивные элементы

Элемент Описание
form Общий контейнер для сбора и отправки данных.
input type="text" Однострочное поле для ввода текста.
input type="password" Поле для ввода пароля (символы скрываются).
input type="radio" Радиокнопка (выбор одного варианта).
input type="checkbox" Флажок (выбор нескольких вариантов).

4. Таблицы

Тег Описание
table Основной контейнер таблицы.
tr Строка таблицы.
th Заголовочная ячейка.
td Обычная ячейка с данными.
